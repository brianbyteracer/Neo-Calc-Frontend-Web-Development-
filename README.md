# NeoCalc

NeoCalc is a modern and responsive front-end calculator built using HTML, CSS, and JavaScript. It performs basic arithmetic operations through a clean, user-friendly interface while showcasing core web development concepts such as DOM manipulation, event handling, and responsive design.

## Features
- Addition, subtraction, multiplication, and division
- Responsive design for desktop and mobile
- Clean and modern UI
- Real-time calculations
- Lightweight and easy to use

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Learning Outcomes
This project helped strengthen my understanding of:
- Front-end development
- DOM manipulation
- Event handling
- Responsive web design
- JavaScript fundamentals
